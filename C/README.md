# Blockchain-Terminologies - C

<p align="center">
<img src="https://1.bp.blogspot.com/-L2y90BbBkdY/X-2udWf7BhI/AAAAAAAAAyY/1wmy71mP_oY01zwf1w6snXiwGM5q0pNJACLcBGAsYHQ/s320/BCT-C.jpeg" width="170" align="center">
</p>

- **Cryptography** is the art and science bof creating and solving codes for securing communication between intended parties.
<p align="center">
<img src="https://1.bp.blogspot.com/-SwOnxUTNI9M/X-2udS3h2zI/AAAAAAAAAyc/xq8y4RteWS8ebqLXbN-1k_A_YKVutvRTgCLcBGAsYHQ/s805/BCT-C1.jpeg" width="220" align="center">
</p>

- **Confidentiality** is one of the objectives in the CIA model of Information Security. It is an assurance that the data in transmission or on storage is not disclosed to unauthorized entities. Encryption is the method to ensure confidentiality.
<p align="center">
<img src="https://1.bp.blogspot.com/-Hg54uiSyVTo/X-2ueBVAQAI/AAAAAAAAAyg/j0AZwXuzOvUKhgvXmcs79vBCuo2VDCeJQCLcBGAsYHQ/s961/BCT-C2.jpeg" width="410" align="center">
</p>

- **Cryptographic Hash Function** is a one way mathematical function which takes an arbitrary length input and provides a fixed length output called hash value. MD and SHA are the most popularly known family of Hashing algorithm.
<p align="center">
<img src="https://1.bp.blogspot.com/-0EsdRfk3avU/X-2ueRZbDeI/AAAAAAAAAyk/zs69vh2LOkkiwx0Ix1X_McvwmXftVXuwACLcBGAsYHQ/s952/BCT-C3.jpeg" width="420" align="center">
</p>

- **Cryptocurrency** is a digital virtual money used as a medium of exchange where creation, transfer, ownership are recorded in a distributed ledger using strong cryptographic algorithm.
<p align="center">
<img src="https://1.bp.blogspot.com/-ZoLy2bK6lAM/X-2uedc8AhI/AAAAAAAAAyo/maQX3uyqnlQD3exfMd3HrlmEUDLSgS0RwCLcBGAsYHQ/s868/BCT-C4.jpeg" width="220" align="center">
</p>

- **Cryptoasset** is a digital representation of real-world asset or value, real or virtual collectible that is created, stored and traded in a distributed ledger using strong cryptographic algorithm.
<p align="center">
<img src="https://1.bp.blogspot.com/-e6p0JSfO6o8/X-2ue9VlESI/AAAAAAAAAys/Jki_PCeK6v0ZZ_7QHD6DKeVZL2em8oRLgCLcBGAsYHQ/s750/BCT-C5.jpeg" width="310" align="center">
</p>

- **Consensus** is a fault-tolerant method through which all the peers in the distributed network agree on the present state of the ledger.
<p align="center">
<img src="https://1.bp.blogspot.com/-h1dJlbEkdng/X-2ufHw8i3I/AAAAAAAAAyw/prRlXZJsnHU438AXSmTrGxQXHiwgQkhDQCLcBGAsYHQ/s521/BCT-C6.jpeg" width="200" align="center">
</p>

- **Confirmed Block** is a block with valid transactions that is part of the Blockchain Ledger after successful Consensus.
<p align="center">
<img src="https://1.bp.blogspot.com/-wOUiN_A3JGk/X-2ufNiKS3I/AAAAAAAAAy0/9QEoIDZQTu8cXL8v1Nx0dzzODzgc8WaygCLcBGAsYHQ/s779/BCT-C7.jpeg" width="230" align="center">
</p>

- **Coinbase Transaction** is a unique and first transaction in a block that is created by a miner to collect the block reward and the transaction fees.
<p align="center">
<img src="https://1.bp.blogspot.com/-3Z2ksAzgULY/X-2ufv_ibrI/AAAAAAAAAy4/buJQql3vzv4qvrPYMdN_bM_plbfbFNHwgCLcBGAsYHQ/s535/BCT-C8.jpeg" width="200" align="center">
</p>

- **Chaincode** is a program that contains the business logic written to read and update the ledger state. Chaincode is written in Go, node.js or Java and runs in a secured docker container.
<p align="center">
<img src="https://1.bp.blogspot.com/-4jyENsq3zK4/X-2uf83fHVI/AAAAAAAAAy8/Rq9yDzGtyxcWD3ER_QOeify86GnDMOZagCLcBGAsYHQ/s464/BCT-C9.jpeg" width="180" align="center">
</p>

- **Consortium Blockchain** or Federated Blockchain is a permissioned Blockchain network across multiple organisations.
<p align="center">
<img src="https://1.bp.blogspot.com/-6JXTZIsMX0g/X-2udTgLyKI/AAAAAAAAAyU/EqYYzbC0ZEcNVPUkCij-Gw32WzTmTbzfQCLcBGAsYHQ/s719/BCT-C10.jpeg" width="300" align="center">
</p>
